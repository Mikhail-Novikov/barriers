# Описание проекта

Проект представляет собой веб-приложение

# Технологии
  HTML, CSS, JavaScript, Pug, Gulp, Webpack, Babel.

# Структура проекта
  Проект организован следующим образом:
    src/pug - находятся Pug-шаблоны для генерации HTML.
    src/scss - находятся SCSS-стили для стилизации проекта.
    src/js - находятся JavaScript-файлы для реализации логики проекта.
    gulpfile.babel.js - определены задачи для сборки проекта.
    package.json - указаны зависимости и конфигурация проекта

## Использование

Чтобы запустить локальный сервер разработки, выполните команду:

```bash
node v.8.17.0
npm i
gulp
```
